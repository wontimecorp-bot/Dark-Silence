# SDD Pilot - Governance Files

`project-instructions.md` and `.github/sddp-config.md` are governance documents. Edit them via:

- `.github/skills/init-project/SKILL.md` - bootstrap or amend principles, technology stack, and config
- `.github/skills/instructions-management/SKILL.md` - versioning, amendment changelog, propagation policy
- `.github/skills/amend-project/SKILL.md` - propagate changes across PRD / SAD / DOD / project-plan

Preserve all existing template placeholders and section headings. Amendments require a version bump with an ISO-dated changelog entry per `project-instructions.md` Governance section.
