[Completeness, Plan §Error Handling, Contracts §Message catalog] <!-- Evaluator: Resolved — added TR-030 (decode-fail/unknown/truncated/oversize → drop, no state mutation, log; not clamped) to spec.md; split plan Error Handling into a dedicated malformed-packet row -->
